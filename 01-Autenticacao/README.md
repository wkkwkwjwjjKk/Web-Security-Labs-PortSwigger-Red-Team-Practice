# 🔐 Authentication & Access Control

Categoria dedicada a vulnerabilidades em autenticação e controle de acesso.

## Subcategorias

- **2FA Broken Logic** - Bypass de 2FA via manipulação de estado
- **Authentication Vulnerabilities** - Falhas gerais em login
- **Password-Based Login** - Exploração de fluxo de senha
- **JWT Authentication** - Bypass de JWT e manipulação de token

## Conceito-Chave

**Estado no Servidor**: Autenticação crítica deve ser gerenciada pelo servidor, NUNCA confiar em cliente.
