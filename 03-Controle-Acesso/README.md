# 🔓 Access Control

Categoria dedicada a vulnerabilidades em autorização e controle de acesso.

## Temas

- **IDOR** - Insecure Direct Object Reference
- **Privilege Escalation** - Vertical e horizontal
- **Authorization Bypass** - Falhas de validação

## Conceito-Chave

**Server-Side Always**: Validar autorização no servidor em TODA requisição. Nunca confiar em parâmetros do cliente.
