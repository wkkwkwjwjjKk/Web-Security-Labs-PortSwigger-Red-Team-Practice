🛡️ PortSwigger Labs – Web Security Writeups

Este repositório contém meus estudos práticos de Web Security através da PortSwigger.

O foco é desenvolver habilidades em exploração de vulnerabilidades reais em aplicações web, com ênfase em segurança ofensiva e análise de falhas.

🎯 Objetivo

Construir base prática em Web Exploitation e evolução técnica em Red Team.
