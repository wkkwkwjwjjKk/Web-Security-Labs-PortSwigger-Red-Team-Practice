# Web Security Labs

## Vulnerabilidades estudadas

### SQL Injection
- Authentication bypass
- UNION attacks
- Blind SQL Injection
- XML SQLi
- Conditional responses
- Time-based SQLi

### SSRF
- Basic SSRF
- SSRF with filter bypass
- SSRF against localhost
- SSRF with open redirect

### Access Control
- IDOR
- Role escalation
- Unprotected admin functionality
- Parameter-based access control
- Multi-step process vulnerabilities

### Command Injection
- Basic OS command injection
- Blind command injection
- Time delays
- Out-of-band exfiltration

### File Upload Vulnerabilities
- Web shell upload
- Extension bypass
- Content-Type bypass
- Path traversal upload

### Authentication Vulnerabilities
- Username enumeration
- Broken brute force protection
- MFA vulnerabilities
- Response timing attacks

## Ferramentas utilizadas

- Burp Suite
- Repeater
- Intruder
- Hackvertor
- DevTools

## Objetivo

Desenvolvimento de habilidades práticas em:
- Web Application Security
- Pentest Web
- HTTP Analysis
- Exploração manual de vulnerabilidades
- Enumeração
- Bypass de filtros
