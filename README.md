# PortSwigger Labs – Web Security Writeups

Este repositório contém writeups de laboratórios da PortSwigger Web Security Academy, focados em vulnerabilidades de segurança em aplicações web.

O objetivo é documentar aprendizado prático em Web Exploitation e Segurança Ofensiva.

---

## 📌 Áreas estudadas

### Command Injection

* Basic Command Injection
* Time-based (Delay) Command Injection
* Blind OS Command Injection with Output Redirection

---

### File Upload Attacks

* Remote Code Execution via Web Shell Upload
* Content-Type Restriction Bypass
* Path Traversal Upload
* Extension Blacklist Bypass
* Obfuscated File Extension

---

## 🧠 Foco de estudo

* Web Application Security
* Exploitation de vulnerabilidades reais
* Técnicas de Remote Code Execution (RCE)
* Bypass de validações de segurança

---

## 🛡️ Conceitos abordados

* Command Injection
* File Upload Vulnerabilities
* Web Shells
* Input Validation Bypass
* Path Traversal

---

## 🎯 Objetivo

Construir base prática sólida em Web Security para atuação futura em Red Team e Pentesting.
