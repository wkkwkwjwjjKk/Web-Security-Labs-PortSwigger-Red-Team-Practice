# 🔗 APIs & Web Security

Categoria dedicada a vulnerabilidades em APIs REST modernas.

## Subcategorias

- **API Security & Web Exploitation** (5 labs) - Endpoints ocultos, mass assignment, parameter pollution
- **OAuth & OpenID Labs** - Bypass de OAuth, account linking

## Conceito-Chave

**Whitelist Always**: Negar por padrão, apenas permitir explicitamente. APIs herdam vulnerabilidades web + suas próprias.
