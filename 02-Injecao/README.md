# 💉 Injection Attacks

Categoria dedicada a exploração de injeção de código em várias formas.

## Subcategorias

- **SQL Injection** (9 labs) - Desde UNION básico até WAF bypass
- **Blind SQL Injection** (3 labs) - Boolean, time-based, OAST
- **Command Injection** - Execução de comandos do SO
- **XXE Injection** (8 labs) - XML External Entity em todas as variantes

## Conceito-Chave

**Dados ≠ Código**: Sempre separar entrada do usuário de lógica de programação. Use prepared statements, não concatenação.
